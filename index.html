Tentang Kami

Asisten Laboratorium Teknik Informatika

Visi

Menjadi pendukung kegiatan mahasiswa dengan perkuliahan berbasis praktik pada Program Studi Informatika Universitas Pamulang, guna mendorong kemajuan belajar dan inovasi di lingkungan kampus

Misi

  • Memberikan dukungan teknis dan sumber daya perkuliahan berbasis praktik yang dibutuhkan oleh mahasiswa dan dosen.
  • Membangun komunitas belajar yang ramah, di mana setiap anggota dapat saling berbagi pengetahuan.
  • Menyelenggarakan pelatihan dan workshop untuk meningkatkan keterampilan teknis dan profesionalisme anggota.
  • Menyediakan akses mudah untuk informasi terkait penggunaan lab dan kegiatan perkuliahan berbasis praktikum.
  • Aktif berkontribusi menjalin kerjasama pada berbagai kegiatan dalam kegiatan program studi informatika.

Mata Kuliah

Deskripsi singkat Mata kuliah berbasis praktik

Algoritma dan Pemrograman

Mata kuliah Algoritma dan Pemrograman menggunakan C++ memberikan pengantar tentang bahasa C++, struktur kontrol, fungsi, array dan string, pointer dan referensi, struktur data, algoritma dasar, serta analisis algoritma. Selain itu, juga mencakup pemecahan masalah dan praktikum untuk mengasah keterampilan pemrograman mahasiswa. Praktikum ini bertujuan untuk menerapkan konsep-konsep yang dipelajari dalam pembelajaran teoritis menjadi keterampilan praktis dalam pemrograman menggunakan C++.

Download Modul

Struktur Data

Mata kuliah Struktur Data dengan C++ membahas pemahaman dan implementasi struktur data dalam bahasa C++. Mahasiswa mempelajari array, linked list, stack, queue, tree, dan graph, serta teknik operasionalnya. Materi mencakup sintaksis, implementasi, analisis efisiensi algoritma, dan penerapan dalam pemrograman. Praktikum melibatkan latihan pemrograman C++ untuk memperkuat keterampilan praktis dalam menggunakan struktur data.

Download Modul

Data Mining

Mata kuliah Data Mining memperkenalkan konsep, teknik, dan alat untuk mengekstraksi pengetahuan dari data besar. Mahasiswa mempelajari klasifikasi, klastering, asosiasi, prediksi, preprocessing data, evaluasi model, dan penerapan praktis dalam berbagai domain. Materi mencakup algoritma, perangkat lunak, etika, dan privasi data. Praktikum melibatkan proyek analisis data nyata.

Download Modul

Basis Data

Mata kuliah Basis Data memberikan landasan penting dalam pengembangan web. Mahasiswa mempelajari konsep dasar basis data, model data, dan bahasa pemrograman basis data seperti SQL. Pengetahuan ini penting dalam merancang sistem manajemen basis data yang efisien untuk aplikasi web, memastikan keamanan data, meningkatkan kinerja situs web, dan memenuhi kebutuhan pengguna dengan baik.

Download Modul

Jaringan Komputer

Mata kuliah Jaringan Komputer membahas konsep dasar jaringan, protokol komunikasi, dan teknologi terkait. Mahasiswa mempelajari struktur jaringan, protokol seperti TCP/IP, dan teknik routing serta switching. Mata kuliah ini penting untuk memahami internet, keamanan jaringan, dan aplikasi teknologi jaringan dalam pengembangan web dan sistem terdistribusi. Dengan penguasaan ini, mahasiswa dapat merancang, mengelola, dan memelihara jaringan komputer dengan efisien, serta mengimplementasikan solusi inovatif dalam pengembangan teknologi informasi.

Download Modul

Cloud Computing

Mata kuliah Cloud Computing membahas konsep, arsitektur, dan teknologi terkait penggunaan sumber daya komputasi melalui internet. Mahasiswa mempelajari dasar-dasar cloud computing, jenis layanan (IaaS, PaaS, SaaS), arsitektur, manajemen sumber daya cloud, virtualisasi, layanan populer (AWS, Azure, GCP), keamanan, hukum, desain, dan implementasi aplikasi cloud. Praktikum melibatkan eksperimen dengan layanan cloud dan pengembangan aplikasi menggunakan berbagai teknologi dan platform.
